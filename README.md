# LLMs_as_Biomedical_Annotators

This repository contains all of our tasks related to our paper regarding using LLMs as Biomedical Relation annotators.

## Getting Started

The resources for this project exceed GitHub's file size limit, even after compression. As a solution, we have hosted all the Benchmarks on Dropbox.

### Download Resources

[Download Notebooks and Results from Dropbox](https://www.dropbox.com/scl/fi/sg4wqxeshba4zmgackufl/LLM_benchmarks.zip?rlkey=2i4613zox7xvdvh96j3sa3tyb&st=2b961i4d&dl=0)

After downloading, extract the files into the repository folder:
```bash
unzip LLM_benchmarks.zip
```
